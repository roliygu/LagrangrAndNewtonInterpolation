LagrangrAndNewtonInterpolation
==============================

LagrangrAndNewtonInterpolation
拉格朗日插值和牛顿插值的C++代码，欢迎使用检验。
插值作用：从给定点的坐标得到一条拟合的曲线函数。
拉格朗日差值与牛顿插值的共同点：给定n个点的左边可以唯一确定一条n-1次函数，两者本质上是等价的。
不同：牛顿插值是拉格朗日的优化，在于新加点坐标增加的计算量远小于拉格朗日插值。
缺点：给定的点过多会得到高次函数，产生龙格现象（高次震荡），而产生大误差。
函数说明：
	Lagrange参数为（x，y）的数组，个数n。计算公式参考教材；
	Newton参数是x的一维数组和y的二维数组，y需要打一张差商的表，函数内的第一个for循环即在打表。计算公式参考教材。
